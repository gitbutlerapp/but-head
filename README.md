# ButHead

A custom monospace font with a unique design philosophy and interactive showcase website.

![ButHead Font Cover](https://gitbutler-docs-images-public.s3.us-east-1.amazonaws.com/but-head-font.webp)

## Overview

ButHead is a display font with monospace character, available in Regular and Italic styles. Designed with distinctive letterforms for visual impact and creative typography. The project includes the font source files and a demo website built with Svelte.

## Project Structure

- **font/** - Font development files
  - `source/` - Glyphs source files (ButHead-Regular.glyphs, ButHead-Italic.glyphs)
  - `otf/` - OpenType font files
  - `web/` - Web font files and CSS
- **pages/** - Interactive showcase website (Svelte + Vite)

## Demo

View the live demo at: https://gitbutlerapp.github.io/but-head/

### Development

```bash
cd pages
pnpm install
pnpm run dev
```

### Build

```bash
cd pages
pnpm run build
```
